## Overview

The [lastfm-bulk-edit](https://github.com/RudeySH/lastfm-bulk-edit) userscript by RudeySH adds bulk editing capabilities to Last.fm.

## Requirements

- Last.fm Pro subscription (required for any scrobble editing)
- Userscript manager (e.g., Tampermonkey, Violentmonkey, Greasemonkey)

## Features

- Adds "Edit scrobbles" menu items for bulk operations
- Edit multiple tracks, albums, or artists at once
- Manage automatic edits more conveniently
- Useful for correcting artist names to composer names across many scrobbles

## Installation

1. Install a userscript manager browser extension
2. Visit the <a href="https://github.com/RudeySH/lastfm-bulk-edit" target="_blank">GitHub repository</a>
3. Follow installation instructions in the README

## Use Case: Artist to Composer Correction

To change scrobbles from performer to composer:

1. Navigate to your Last.fm library
2. Use the bulk edit menu to select tracks
3. Change the Artist field to the composer's name
4. Enable "Automatic edit" to apply to future scrobbles of the same track
