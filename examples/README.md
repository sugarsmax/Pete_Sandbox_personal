# Example Notebooks

Working examples demonstrating common patterns and workflows.

## Structure

Organize by topic or project:
- `music_analysis/` — Last.fm and music data analysis
- `data_processing/` — CSV/data manipulation examples
- `web_scraping/` — HTTP requests and web scraping patterns
- Add subdirectories as needed

## Usage

Each example notebook is:
- Self-contained with explanatory cells
- Runnable end-to-end
- Documented with assumptions and setup steps

Start with notebooks in your area of interest and adapt for your needs.
